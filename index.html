CSS Filters

CSS Filters are used to add visual effects to the elements of an image such as blur or color shift and are applied as the last attribute the web page loads. These are wonderful tools to enhance the image(s) in order to further convey their purpose or overall design of the webpage. It is important; however, to note that while they are a powerful tool, they can have an impact on the performance of the site. Most filters will have very minor impact, but others with any blurring tend such as ‘blur’ or ‘drop-shadow’ are known to be slower than others. This is due to the browser needing to calculate exponentially when mixing the colored pixels, causing the loading process to happen that much slower. This does not mean that one should stay away from using these tools, but to rather be aware in case adjustments need to be made for different platforms and devices.

Image Hover Effects

Image Hover Effects are exactly what they sounds like. They are effects that are applied when a user hovers their cursor over an image. These can be effects such as rotating or changing color filters, showing text, or highlighting a link. It is important; however, to make sure there is purpose when adding these effects, “[m]ake it easier for users to differentiate between a text and link, a fancy round corner text box and a call-to-action button”. On a webpage, the purpose is to clearly convey the information to the user and adding an overdose of these effects can cause the inverse of its purpose and end up confusing the user instead.
